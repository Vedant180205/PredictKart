## 🏗️ HOW TO DEPLOY THIS BEAST

### 1. CLONE THE SOUL OF THE MACHINE
```bash
git clone https://github.com/your-repo/prerdict-cart.git
cd prerdict-cart
```

### 2. FUEL THE BACKEND
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Or `venv\Scripts\activate` on Windows
pip install -r ../requirements.txt
uvicorn main:app --reload --port 8000
```

### 3. IGNITE THE FRONTEND
```bash
cd ../frontend
npm install
npm run dev
```

### 4. CONFIGURE THE SECRETS (.env)
You need the keys to the kingdom:
```env
SCRAPERAPI_KEY=your_key_here
GROQ_API_KEY=your_key_here
SERPAPI_KEY=your_key_here
DATABASE_URL=your_db_connection_string
```
